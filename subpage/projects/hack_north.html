Home Automation System
Home Automation System
Sep. 2014
Hackathon of Hack the North, University of Waterloo

Designed a home automation system monitored by Arduino and Spark processor. Responsible for MCU programming, circuit design, and UI design.
User is able to control the home devices remotely with the Android application by speaking to the phone. We made use of the wit.ai for the NLP interface.
